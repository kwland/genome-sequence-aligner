# Open The Website In Microsoft Edge

Use this on Windows:

1. Open the project folder.
2. Double-click `open_in_edge.bat`.
3. Microsoft Edge should open the app at:

```text
http://localhost:8000
```

Keep the black command window open while using the website. Closing it stops the local website.

If the launcher says Python was not found, open `index.html` directly with Microsoft Edge instead.
